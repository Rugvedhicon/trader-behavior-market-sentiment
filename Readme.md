# Trader Behavior vs Market Sentiment Analysis

This project analyzes how Bitcoin market sentiment (Fear & Greed Index) 
influences trader behavior and performance using historical trading data.

## Datasets Used
- Bitcoin Fear & Greed Index (daily market sentiment)
- Historical trader data from Hyperliquid

## Objective
To understand how different market sentiment conditions affect:
- Trader profitability
- Trading activity
- Risk-taking behavior

## Key Insights
- Traders tend to be more active and profitable during Greed periods.
- Greed is associated with higher risk-taking and trade frequency.
- Fear periods show fewer trades but deeper losses, indicating panic-driven behavior.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
